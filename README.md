🔍 Port Scanner

A simple Python script that scans random ports (1–9999) on a target to find open ones.

🚀 Usage
python scanner.py

Enter a target when prompted:

Enter target (IP or domain): example.com
⚙️ Features
Scan ports 1–9999
Randomized scanning order
Detects open TCP ports
No external libraries needed
⚠️ Disclaimer

For educational use only. Do not scan systems without permission.
